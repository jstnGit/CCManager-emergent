## üì± App Overview

### What is this app?
A Flutter mobile application for tracking credit card expenses with offline-first architecture using local device storage.

### Key Technologies
- Flutter 3.0+
- Dart 3.0+ (null-safe)
- Hive (local database)
- Provider (state management)
- Material 3 (UI design)

### Main Features
1. Credit card dashboard with real-time tracking
2. Add, edit, delete expenses
3. Mark expenses as paid/unpaid
4. Statement and due date management
5. Offline-first (no internet required)

### Platform Support
- ‚úÖ Android
- ‚úÖ iOS
- ‚úÖ Offline mode
